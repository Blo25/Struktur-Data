## Sistem Pengelolaan Nilai Mahasiswa Menggunakan Array
## 1. Penjelasan konsep array
Array adalah struktur data yang digunakan untuk menyimpan banyak 
data dengan tipe yang sama dalam satu variabel.
Dalam Python, array biasanya direpresentasikan menggunakan list. 

Keuntungan menggunakan array :
* Memudahkan penyimpanan banyak data
* Memudahkan proses perulangan
* Mempermudah pengolahan data
